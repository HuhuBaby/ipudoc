# 配置文件介绍

#配置说明

|配置项|说明|

|-----|----|

|database|根节点|


|ipu(连接别名)|数据库连接别名(特别注意：在创建操作数据库的Dao的实例时，需要传入此参数)|

|type|连接类型|

|driver|数据库驱动类名|

|url|数据库的统一资源定位符|

|user|用户名|

|passwd|密码|

|initialSize|初始化连接数|

|maxActive|最大连接数据|

|maxIdle|最大空闲连接|

|maxWait|超时等待时间(以毫秒为单位)|



#配置范例

```xml

<?xml version="1.0" encoding="UTF-8"?>

<database>

    <ipu

        type="dbcp"

        driver="oracle.jdbc.driver.OracleDriver"

        url="jdbc:oracle:thin:@10.7.5.75:1521:ncrmdb"

        user="bband"

        passwd="bband-123"

        initialSize="5"

        maxActive="20"

        maxIdle="5"

        maxWait="1000"

        />

</database>

```
